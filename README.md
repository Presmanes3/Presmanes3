Machine Learning Engineer @ CERN
--------------------------------

I’ve been working as a Software Machine Learning Engineer at CERN, building end-to-end, real-time inspection systems for superconducting cable manufacturing. I architected a distributed, containerized microservices platform (Python, Docker, Kafka, Celery/Redis) processing ~2 GB/min of multi-sensor data with sub-second latency, operating 8–10h shifts and ~1 TB/day. I productionized anomaly detection (PatchCore/ResNet, autoencoders) with strong MLOps/observability (MLflow, drift monitoring, Grafana) and reproducible data/model management (PostgreSQL + S3/MinIO, automated backups with Airflow).

* 🌍  I'm based in Prevressin Moens, France
* ✉️  You can contact me at [presmanes3@gmail.com](mailto:presmanes3@gmail.com)
* 🚀  I'm currently working on [ConcepTracker](http://github.com/Presmanes3/ConcepTracker)
* 👥  I'm looking to collaborate on Machine Learning Projects
